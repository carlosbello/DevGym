Buscar en la lista

Escriba una función llamada find que reciba un elemento a buscar y una lista, y devuelva el índice del elemento en la lista. En caso de que el elemento no esté en la lista, deberá devolver -1.

Ejemplo:

> find('hola', ['Este', 'es', 'un', 'ejemplo'])
-1            
> find(2, [1, 3, 5, 2, 8, 2, 9])
3
